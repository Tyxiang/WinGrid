# 用途
将当前窗口调整为特定大小并放置在特定位置。

# 用法
- 同时按下鼠标左右键触发窗口调整；
- 调整结果取决于触发时鼠标指针位于窗口的哪个区域；

# 调整结果
1. 屏幕左侧  
1. 屏幕右侧  
1. 屏幕左上  
1. 屏幕左下  
1. 屏幕右上  
1. 屏幕右下
