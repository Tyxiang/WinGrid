# 用途
将当前窗口调整为特定大小并放置在特定位置。

# 用法
- 同时按下鼠标左右键触发窗口调整；
- 调整结果取决于触发时鼠标指针位于窗口的哪个区域；

# 窗口区域
<table>
  <tr>
    <td>L</td><td>R</td>
  </tr>
</table>

</table>

# 调整结果
1.屏幕左侧  
2.屏幕右侧  
3.屏幕左上  
4.屏幕左下  
5.屏幕右上  
6.屏幕右下
