# 用途
将当前窗口调整为特定大小并放置在特定位置。

# 用法
- 同时按下鼠标左右键触发窗口调整；
- 调整结果取决于触发时鼠标指针位于窗口的哪个区域；

  <tr>
    <td>调整至屏幕左侧</td><td>调整至屏幕右侧</td>
  </tr>
